# 📰 Knowledge game
**Knowledge Game** is a fullstack educational application that blends fast-paced gameplay with trivia challenges. Built using JavaScript, Express.js, and MongoDB, it offers an engaging way to test knowledge while navigating an interactive game world.

## Features
  - Gameplay: Jump, shoot, and destroy planets.
  - Quiz integration: Answer trivia questions to progress.
  - Real-time scoreboard: Track high scores.

## Tech Stack
* Vanilla JavaScript
* Express.js
* MongoDB
